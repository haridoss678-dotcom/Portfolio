# HARIDOSS B Portfolio

A premium AI & Machine Learning Engineer portfolio homepage built with HTML5, CSS3, and Vanilla JavaScript.

## Files

- `index.html` — homepage structure and content
- `style.css` — responsive visual styling and layout
- `script.js` — typing animation, mobile navigation, scroll reveal, and form feedback
- `server.py` — local Python server using only the standard library
- `assets/icons/` — local SVG icon assets
- `assets/images/` — profile image and project placeholders
- `assets/resume/Haridoss_Resume.pdf` — placeholder resume file

## Run locally

1. Open a terminal in the `portfolio` folder.
2. Run:

```bash
python server.py
```

3. Open your browser to `http://localhost:8000`.
